Workout Wizard
Workout Wizard is a comprehensive fitness companion designed to make health and wellness accessible, transformative, and enjoyable. This project provides a suite of tools including a nutrition tracker and a metabolic activity calculator to help users unlock their full potential.

🚀 Features
🥗 Nutrition Tracker
A dynamic tool that allows users to calculate the nutritional value of their meals.

Extensive Database: Includes data for 25+ common foods such as fruits (apples, bananas), proteins (chicken, beef, lentils, tofu), and grains (rice, quinoa).

Macro breakdown: Provides detailed calculations for Total Calories, Protein, Fat, and Carbohydrates based on gram-specific input.

Dynamic UI: Add multiple ingredients to a single meal to get a cumulative nutritional summary.

🏃 Calories Burned Calculator
An activity-based calculator to estimate energy expenditure during various workouts.

Activity Selection: Supports tracking for Running, Cycling, Swimming, Walking, and Jumping Rope.

MET-Based Accuracy: Uses Metabolic Equivalent of Task (MET) values to calculate burn rates relative to the user's weight and the duration of the activity.

🤝 Community & Support
Workout Wizard aims to foster a supportive environment through:

Expert Guidance: Dedicated to providing knowledge from certified trainers and wellness advocates.

Innovative Programs: A diverse range of offerings including HIIT, yoga, and meditation.

🛠️ Technical Stack
HTML5: Structured content for the main application and "About Us" pages.

CSS3: Custom styling for a clean, responsive user interface.

JavaScript (ES6): Powering the logic for nutritional calculations and metabolic formulas.

📁 Project Structure
activitycal.html / .js / .css: The dedicated module for calculating calories burned during physical activities.

app.js: The core logic for the nutrition tracking database and calculation engine.

about-us.html: Detailed information regarding the mission and offerings of Workout Wizard.
